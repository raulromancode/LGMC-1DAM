Los tres pilares de la Web

Tim Berners-Lee es considerado el padre de la Web porque desarrollo los tres elementos basicos para el funcionamiento de la Web

  • HTML
  • HTTP
  • URL

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Fue desarrollado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo de los años. HTML utiliza una serie de elementos y etiquetas para estructurar el contenido en la web.

Fuente: HTML

HTTP

HTTP (HyperText Transfer Protocol) es el protocolo de comunicación utilizado para la transmisión de información en la Web. Fue desarrollado por Tim Berners-Lee y es la base de la comunicación en la World Wide Web.

Fuente: HTTP

URL

URL (Uniform Resource Locator) es la dirección que se utiliza para acceder a un recurso en la Web. Fue desarrollado por Tim Berners-Lee y es fundamental para la navegación en Internet.

Fuente: URL
