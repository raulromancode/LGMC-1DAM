Mi sitio web

Bienvenido a mi sitio web. Aquí encontrarás información sobre los tres pilares de la Web: HTML, HTTP y URL.

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web.

HTTP

HTTP (HyperText Transfer Protocol) es el protocolo de comunicación utilizado para la transmisión de información en la Web.

URL

URL (Uniform Resource Locator) es la dirección que se utiliza para acceder a un recurso en la Web.
